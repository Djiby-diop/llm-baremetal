# Contributing to Operating Organism

This repository contains a **public prototype** of the Operating Organism project.

The full implementation is maintained in a private research repository.

## What you can do

- **Open issues** for bugs in the prototype code
- **Discuss** the architecture concepts in Discussions
- **Fork** and experiment with the zone allocator or BLAS kernels

## What this repo is NOT

This is not the production codebase. Advanced components (TLS oracle, federation 
protocol, DIOP model, organ bus) are maintained separately.

## Contact

For research collaboration: open an issue or reach the author via AUTHORS file.
